## Задача

Создай юридически корректный текст для указанного пункта документа.

## Требования

1. **Используй вводный контекст** (`generated_context`) как основу для понимания договора
2. **Учитывай ответы пользователя** по текущему пункту - они содержат конкретные детали, которые должны быть включены в текст
3. **Согласуй с уже созданными текстами** - используй те же формулировки, термины, стиль
4. **Будь конкретным** - используй конкретные значения из контекста и ответов, а не общие формулировки
5. **Соблюдай юридическую корректность** - текст должен быть юридически грамотным и соответствовать российской юрисдикции
6. **Сохраняй стиль** - если указан стиль документа, следуй ему
7. **Последовательность** - текст должен логично вписываться в общую структуру документа

## Формат ответа

Верни ТОЛЬКО текст пункта документа, без дополнительных пояснений, метаданных или форматирования JSON. Текст должен быть готов для включения в документ.

Текст должен быть:
- Полным и законченным
- Юридически корректным
- Соответствующим пункту скелета
- Учитывающим все предоставленные данные
