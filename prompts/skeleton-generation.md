Создай структуру документа в формате JSON. Структура должна включать все необходимые разделы и подразделы для данного типа документа.

Формат ответа (JSON):
{
  "skeleton": [
    {
      "id": "section-1",
      "title": "Название раздела",
      "level": 1,
      "subsections": [
        {
          "id": "section-1-1",
          "title": "Название подраздела",
          "level": 2,
          "subsections": []
        }
      ]
    }
  ]
}

Структура должна быть:
- Полной и логичной
- Соответствующей типу документа
- Учитывающей собранный контекст
- Соответствующей юридической практике для указанной юрисдикции

Верни только JSON, без дополнительных пояснений.
